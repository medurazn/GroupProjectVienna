Purpose Page

For this project, we decided to showcase the city of Vienna. The purpose of this project is to demonstrate what we have learned in class so far, and implementing it. As one can see in the html text files, in-line css was used. The website consists of 6 pages: a main page, 2 outstanding landmark pages, a sports page, a photo gallery and the purpose page. All of the pages are linked and open in the same window. External links were also used as well as section links. The content of the website has all been formatted utilizing various styles and attributes. Various pictures were used,  just wished we could have organized them a little better. We included background music, a hoverable-horizontal navigation bar and also added a video clip in one of our pages. The picture gallery was organized using a table. More practice is required in order to fully utilize Expressions Web.
